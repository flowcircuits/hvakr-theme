# HVAKR Themes for Visual Studio Code

Light and dark color themes built from the [HVAKR](https://hvakr.com) design system.

## Themes

### HVAKR Light

A clean light theme with warm orange accents, deep blues for functions, and rich greens for strings.

### HVAKR Dark

A dark theme built on deep blue-grays with soft pastels — mint greens, sky blues, and lavender purples.

## Color Palette

Both themes are built on the [Blueprint.js](https://blueprintjs.com) color palette with HVAKR's signature orange (`#fd491a`) as the primary accent.

| Role       | Light             | Dark               |
| ---------- | ----------------- | ------------------ |
| Background | `#ffffff`         | `#111418`          |
| Foreground | `#111418` (95%)   | `#ffffff` (95%)    |
| Accent     | `#fd491a`         | `#fd491a`          |
| Keywords   | `#5642a6` indigo  | `#bdadff` lavender |
| Functions  | `#2d72d2` blue    | `#4c90f0` sky blue |
| Strings    | `#165a36` forest  | `#72ca9b` mint     |
| Types      | `#ac2f33` crimson | `#fa999c` rose     |
| Constants  | `#866103` amber   | `#fbd065` gold     |
| Comments   | `#8f99a8` gray    | `#5f6b7c` gray     |

## Install

Search for **HVAKR Themes** in the VS Code Extensions panel, or:

```
ext install flowcircuits.hvakr-themes
```
